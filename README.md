# Stock Market Analyzer

A console-based Stock Market Analyzer built in Python using efficient Data Structures and Algorithms.

## Features
- **Data Structures**: Lists, Dictionaries (Hash Tables), Deque, Heaps.
- **Algorithms**: Hybrid Sorting (Quick Sort + Merge Sort), Multi-key Search, Priority Scoring (Heaps).
- **Analysis**: Sector-wise classification, Trend Analysis (Moving Averages).

## How to Run

1. Navigate to the project directory:
   ```bash
   cd /Users/apple/.gemini/antigravity/scratch/stock_market_analyzer
   ```

2. Run the CLI application:
   ```bash
   python3 main.py
   ```

3. Follow the on-screen menu instructions.
